Description
A clicker game that is about leveling up the circle, with each level becoming progessively harder, requiring more clicks to complete them.

Instructions
Click to start filling the circle.
Hold Click for a second to show how many clicks you have done.

Notes
The game goes on infinitely and there are proceduarly generated colors after Level 100.
I have originally came up with this very simple concept around late 2021 as well as many others and I'd like to remake them.
